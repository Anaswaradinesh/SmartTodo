## Features

- **Add / Edit / Delete tasks** — full task management with inline editing
- **Due dates & priority levels** — High, Medium, Low priority with visual distribution
- **Task filters** — filter by All, Pending, Completed, or High priority
- **Statistics dashboard** — track completion rate, tasks due today, and priority breakdown
- **Dark mode** — toggle between light and dark themes
- **CSV export** — download your tasks as a spreadsheet
- **Persistent storage** — tasks saved in localStorage, survive page refresh
